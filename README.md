# Portfolio

Bu mening shaxsiy portfolio saytim bo'lib, HTML va CSS yordamida yaratilgan. Sayt orqali men o'z ko'nikmalarim, loyihalarim va tajribam haqida ma'lumot bera olaman.

## 🚀 Texnologiyalar

- HTML5
- CSS3

## 📦 O'rnatish

Loyihani lokal muhitda ishga tushirish uchun quyidagi qadamlarni bajaring:

1. Repozitoriyani klonlang:
```bash
git clone https://github.com/rajabovao335-boop/portofiolo.git
```

2. Loyiha papkasiga o'ting:
```bash
cd portofiolo
```

3. Kerakli paketlarni o'rnating:
```bash
npm install
```

4. Saytni brauzerda ochish uchun `index.html` faylini ishga tushiring.

## 📁 Loyiha tuzilishi

```
portofiolo/
├── index.html
├── style.css
└── README.md
```
